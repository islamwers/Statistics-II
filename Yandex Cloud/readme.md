# Cloud Technologies Assignments

В данной папке будут храниться учебные материалы, домашние задания и проектные работы по теме **облачных технологий**.

Основной фокус раздела — практическое освоение облачной инфраструктуры, удалённой разработки, виртуальных машин, JupyterLab, API-сервисов и интеграции языковых моделей в прикладные проекты.

---

## Что будет находиться в этой папке

В этом разделе будут размещаться:

- домашние задания по облачным технологиям;
- инструкции по работе с виртуальными машинами;
- материалы по запуску JupyterLab на удалённом сервере;
- задания по работе с Python в облачной среде;
- проекты с использованием LLM и API;
- примеры архитектурных схем;
- шаблоны отчётов и README;
- дополнительные материалы для самостоятельной работы.

---

## Основные темы

В рамках заданий будут рассматриваться следующие направления:

- создание и настройка виртуальных машин;
- подключение к серверу по SSH;
- работа с Linux-средой;
- настройка Python-окружения и `venv`;
- запуск JupyterLab на виртуальной машине;
- проброс портов через SSH-туннель;
- загрузка и обработка данных в облаке;
- подключение внешних API;
- работа с Yandex Cloud и Yandex AI Studio;
- интеграция LLM в учебные и прикладные проекты;
- развёртывание простых веб-приложений, Telegram-ботов и автоматизаций.

---
