# ✅ Рекомендации по подготовке

При подготовке к экзамену важно не просто выучить определения, а уметь объяснять:

- какую задачу решает технология;
- в какой части облачной архитектуры она применяется;
- какие у неё преимущества и ограничения;
- с какими другими сервисами она взаимодействует;
- как она использовалась в практических работах курса.

Особое внимание стоит уделить связкам технологий:

- `VM → Network → Security Group → Load Balancer`;
- `CLI → YAML → IaC → Terraform`;
- `Docker → Container Registry → Kubernetes`;
- `Object Storage → Data Proc → Hive`;
- `MySQL → Data Transfer → ClickHouse → DataLens`;
- `Cloud Functions → Object Storage → Trigger`.
