## 📌 Основные понятия

### Нулевая и альтернативная гипотезы

Нулевая гипотеза обычно обозначается как:

$$H_0$$

Альтернативная гипотеза обозначается как:

$$H_1$$

Пример:

$$H_0: p = p_0$$

$$H_1: p \ne p_0$$

---

## ⚠️ Ошибки при проверке гипотез

### Ошибка первого рода

Ошибка первого рода возникает, если мы отвергаем верную нулевую гипотезу:

$$\alpha = P(\text{reject } H_0 \mid H_0 \text{ is true})$$

Параметр $\alpha$ называется **уровнем значимости**.

---

### Ошибка второго рода

Ошибка второго рода возникает, если мы не отвергаем ложную нулевую гипотезу:

$$\beta = P(\text{do not reject } H_0 \mid H_0 \text{ is false})$$

---

### Мощность теста

Мощность теста — это вероятность отвергнуть нулевую гипотезу, когда она действительно ложна:

$$\text{Power} = 1 - \beta$$

---

## 📊 Область принятия и область отвержения

Для проверки гипотез используется тестовая статистика.

- **Acceptance Region (AR)** — область, где $H_0$ не отвергается  
- **Rejection Region (RR)** — область, где $H_0$ отвергается  

Для двустороннего z-теста область отвержения имеет вид:

$$RR = \{|Z_{st}| > z_{\alpha/2}\}$$

---

## 🧮 Тест для доли

Для проверки гипотезы о доле:

$H_0: p = p_0$

$H_1: p \ne p_0$

используется z-статистика:

$z_{st} = \frac{\hat{p} - p_0}{\sqrt{\frac{p_0(1-p_0)}{n}}}$

Критическое правило:

$\text{reject } H_0 \text{ if } |z_{st}| > z_{\alpha/2}$

Условия применимости:

$np_0 \ge 10$

$n(1-p_0) \ge 10$

---

## 📈 z-test для среднего при известной $\sigma$

Если дисперсия генеральной совокупности известна, используется z-тест:

$H_0: \mu = \mu_0$

$H_1: \mu \ne \mu_0$

Тестовая статистика:

$z_{st} = \frac{\bar{X} - \mu_0}{\sigma / \sqrt{n}}$

Критическое правило:

$\text{reject } H_0 \text{ if } |z_{st}| > z_{\alpha/2}$

Условия применения:

- генеральная совокупность нормальна;
- или выборка достаточно большая: $n \ge 40$.

---

## 📉 t-test для среднего при неизвестной $\sigma$

Если генеральное стандартное отклонение неизвестно, используется t-тест:

$H_0: \mu = \mu_0$

$H_1: \mu \ne \mu_0$

Тестовая статистика:

$t_{st} = \frac{\bar{X} - \mu_0}{s / \sqrt{n}}$

Критическое правило:

$\text{reject } H_0 \text{ if } |t_{st}| > t_{\alpha/2, n-1}$

Условие применения:

- генеральная совокупность должна быть нормальной.

---

## 📐 Проверка гипотезы о дисперсии

Для нормальной генеральной совокупности можно проверять гипотезу о стандартном отклонении:

$$H_0: \sigma = \sigma_0$$

$$H_1: \sigma \ne \sigma_0$$

Тестовая статистика:

$$\chi^2_{st} = \frac{(n-1)s^2}{\sigma_0^2}$$

Область отвержения:

$$
RR =
\{ \chi^2_{st} < \chi^2_{1-\alpha/2,\,n-1} \}
\cup
\{ \chi^2_{st} > \chi^2_{\alpha/2,\,n-1} \}
$$


---

## 🔎 p-value

p-value — это минимальный уровень значимости, при котором нулевая гипотеза может быть отвергнута.

Для двустороннего z-теста:

$p\text{-value} = P(|Z| > |z_{st}|)$

или:

$p\text{-value} = 2P(Z > |z_{st}|)$

Для t-теста:

$p\text{-value} = P(|T| > |t_{st}|)$

или:

$p\text{-value} = 2P(T > |t_{st}|)$

Интерпретация:

- если $p\text{-value} \le \alpha$, то $H_0$ отвергается;
- если $p\text{-value} > \alpha$, то $H_0$ не отвергается.

---

## 📎 Связь доверительных интервалов и проверки гипотез

Проверка гипотезы:

$H_0: \mu = \mu_0$

против двусторонней альтернативы:

$H_1: \mu \ne \mu_0$

эквивалентна построению доверительного интервала уровня $1-\alpha$.

Если:

$\mu_0 \notin CI_{1-\alpha}(\mu)$

то нулевая гипотеза отвергается.

Если:

$\mu_0 \in CI_{1-\alpha}(\mu)$

то нулевая гипотеза не отвергается.

## ⚠️ Важные замечания

- Формулировка “не отвергаем $H_0$” не означает, что $H_0$ доказана.
- Уровень значимости $\alpha$ контролирует вероятность ошибки первого рода.
- Мощность теста показывает способность теста обнаружить реальное отклонение от $H_0$.
- p-value не является вероятностью истинности $H_0$.
- Для t-теста важно учитывать число степеней свободы $n-1$.
- Для теста о дисперсии необходимо предположение о нормальности.
