# 📌 Ключевое понятие: критическая точка

Критическая точка уровня $\alpha$ — это число $x_\alpha$, такое что:

$P(X > x_\alpha) = \alpha$

То есть:

- $x_\alpha$ **отсекает правый хвост распределения**
- площадь хвоста равна $\alpha$

Для стандартного нормального распределения:

$P(Z > z_\alpha) = \alpha$

или эквивалентно:

$P(Z < z_\alpha) = 1 - \alpha$

# 📊 Основные темы задания

## 1. Нормальное распределение

Если:

$X \sim \mathcal{N}(\mu, \sigma^2)$

то используется стандартизация:

$Z = \frac{X - \mu}{\sigma}$

Задачи включают:

- нахождение квантилей  
- определение границ интервалов  
- работу с процентными уровнями (например, 5%, 15.87%)  

---

## 2. Распределение Стьюдента

Случайная величина имеет t-распределение:

$T \sim t_k$

если:

$T = \frac{Z}{\sqrt{\chi^2_k / k}}$

где:

- $Z \sim \mathcal{N}(0,1)$  
- $\chi^2_k$ — хи-квадрат распределение  
- $k$ — число степеней свободы  

---

### 📌 Критические точки t-распределения

Таблицы содержат значения $t_{\alpha, k}$ такие, что:

$P(T > t_{\alpha, k}) = \alpha$

Особенности:

- зависят от числа степеней свободы $k$  
- при $k \to \infty$:

$t_k \to \mathcal{N}(0,1)$

---

## 3. Хи-квадрат распределение

Если:

$Y_1, \dots, Y_k \sim \mathcal{N}(0,1)$

и независимы, то:

$\chi^2_k = \sum_{i=1}^{k} Y_i^2$

---

### 📌 Критические точки χ²

Обозначение:

$\chi^2_{\alpha, k}$

такое, что:

$P(\chi^2_k > \chi^2_{\alpha, k}) = \alpha$

---

## 4. Симметричные интервалы

Часто требуется найти интервал:

$P(-a < Z < a) = 1 - \alpha$

или

$P(\mu - \varepsilon < X < \mu + \varepsilon) = \gamma$

где:

$\gamma = 1 - \alpha$
