## 📚 Теоретическая основа

Пусть имеется выборка

$X_1, X_2, \dots, X_n$

из распределения, зависящего от неизвестного параметра $\theta$.

### 1. Функция правдоподобия для дискретных распределений

Если наблюдаются значения $X_1 = x_1, X_2 = x_2, \dots, X_n = x_n$, то функция правдоподобия записывается как

$L(\theta) = P(X_1 = x_1, X_2 = x_2, \dots, X_n = x_n)$.

При независимости наблюдений это выражение принимает вид

$L(\theta) = \prod_{i=1}^{n} P(X_i = x_i)$.

### 2. Функция правдоподобия для непрерывных распределений

Если случайные величины имеют плотность $f(x \mid \theta)$, то

$L(\theta) = \prod_{i=1}^{n} f(x_i \mid \theta)$.

### 3. Логарифмическая функция правдоподобия

Чаще удобнее работать не с $L(\theta)$, а с логарифмом:

$\ell(\theta) = \ln L(\theta)$.

Преимущество логарифма в том, что произведение превращается в сумму:

$\ln \left( \prod_{i=1}^{n} a_i \right) = \sum_{i=1}^{n} \ln a_i$.

Это упрощает дифференцирование и поиск максимума.

### 4. Оценка максимального правдоподобия

Оценка максимального правдоподобия определяется как значение параметра, максимизирующее правдоподобие:

$$\hat{\theta}_{MLE} = \arg\max_{\theta} L(\theta)$$

или эквивалентно

$$\hat{\theta}_{MLE} = \arg\max_{\theta} \ell(\theta)$$

Обычно для нахождения максимума решают уравнение

$\frac{d}{d\theta} \ell(\theta) = 0$.

## 🧮 Полезные логарифмические формулы

При решении задач будут полезны следующие преобразования:

$\ln(ab) = \ln a + \ln b$

$\ln(a^k) = k \ln a$

$\ln(e^k) = k$

$\frac{d}{dx} \ln x = \frac{1}{x}$

$\frac{d}{dx} \ln f(x) = \frac{f'(x)}{f(x)}$

## 📌 Основные свойства точечных оценок

При анализе точечных оценок важно проверять несколько фундаментальных свойств.

### 1. Несмещённость

Оценка $\hat{\theta}$ называется **несмещённой**, если её математическое ожидание совпадает с истинным значением параметра:

$E(\hat{\theta}) = \theta$.

Это означает, что в среднем оценка не завышает и не занижает параметр.

---

### 2. Дисперсия оценки

Разброс оценки относительно её среднего измеряется дисперсией:

$\mathrm{Var}(\hat{\theta})$.

Чем меньше дисперсия, тем точнее оценка.

---

### 3. Эффективность

Если есть две несмещённые оценки $\hat{\theta}_1$ и $\hat{\theta}_2$, то более эффективной считается та, у которой меньше дисперсия:

$\mathrm{Var}(\hat{\theta}_1) < \mathrm{Var}(\hat{\theta}_2)$.

---

### 4. Состоятельность

Оценка называется **состоятельной**, если при увеличении объёма выборки она стремится к истинному значению параметра:

$\hat{\theta}_n \xrightarrow{P} \theta, \quad n \to \infty$.

То есть с ростом размера выборки оценка становится всё точнее.

---

## 📐 Информация Фишера и граница Рао–Крамера

### 1. Информация Фишера

Информация Фишера измеряет, сколько информации о параметре $\theta$ содержится в выборке:

$I(\theta) = -E \left[ \frac{\partial^2}{\partial \theta^2} \ln L(\theta) \right]$.

Для выборки объёма $n$ часто используется обозначение $I_n(\theta)$.

### 2. Граница Рао–Крамера

Для любой несмещённой оценки выполняется неравенство

$\mathrm{Var}(\hat{\theta}) \ge \frac{1}{I_n(\theta)}$.

Если оценка достигает этой границы, то она является **эффективной**.

---

## 📉 Неравенство Чебышёва и состоятельность

Для доказательства состоятельности оценок часто используется неравенство Чебышёва:

$P\left( |\hat{\theta} - \theta| \ge \varepsilon \right) \le \frac{\mathrm{Var}(\hat{\theta})}{\varepsilon^2}$.

Если при $n \to \infty$ дисперсия оценки стремится к нулю, то вероятность заметного отклонения также стремится к нулю, что и даёт состоятельность.

В задачах $11.1$, $11.4$, $11.5$, $11.6$ рассматриваются случайные величины, распределённые по закону Пуассона с параметром $\lambda$ или кратными параметрами вида $2\lambda$, $3\lambda$.

Общая формула вероятности:

$P(X = k) = \frac{\lambda^k e^{-\lambda}}{k!}$.

### 2. Биномиальное распределение

В задачах $11.2$ и $11.3$ рассматриваются вероятности успеха $p$ в независимых испытаниях.

Формула биномиального распределения:

$P(X = k) = \binom{n}{k} p^k (1-p)^{n-k}$.

Необходимо найти оценку параметра $p$:

- методом максимального правдоподобия;
- в дополнениях — методом моментов;
- исследовать свойства естественной оценки доли.

### 3. Геометрическое распределение

В задачах $11.7$ и $11.8$ анализируется число бросков до первого попадания. Это естественно приводит к геометрическому распределению:

$P(X = k) = (1-p)^{k-1}p, \quad k = 1, 2, \dots$

Необходимо:

- вычислить вероятность наблюдаемой серии;
- сравнить две гипотезы о вероятности попадания;
- найти оценку параметра $p$, при которой вероятность серии максимальна;
- в дополнении — исследовать оценку через метод моментов, информацию Фишера и состоятельность.

### 4. Показательное распределение
