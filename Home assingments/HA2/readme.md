# 🎯 Основные темы задания

Домашняя работа проверяет понимание следующих концепций статистики:

### 📚 Основы статистического исследования

- генеральная совокупность (population)
- выборка (sample)
- репрезентативность выборки

---

### 🎲 Методы формирования выборки

- **Simple Random Sampling (SRS)**
- **Stratified Sampling**
- **Cluster Sampling**
- **Systematic Sampling**
- **Quota Sampling**

---

### ⚠️ Ошибки и смещения в данных

- **Coverage error**
- **Selection bias**
- **Nonresponse bias**
- **Measurement error**

---

### 🔬 Дизайн статистического исследования

В заданиях требуется определять:

- **экспериментальные единицы (experimental units)**
- **объясняющие переменные (explanatory variables)**
- **зависимую переменную (response variable)**
- **варианты лечения (treatments)**
- **сопутствующие факторы (confounders)**
