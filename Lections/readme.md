## 🎓 Лекции по математической статистике

В этом разделе собраны **видеолекции и конспекты** по курсу *«Математическая статистика»*.  
Каждая лекция сопровождается кратким описанием ключевых понятий и направлена на формирование системного понимания статистического анализа данных.

---

### 🎯 Цель курса

- Понять философию и задачи статистики.  
- Освоить принципы выборочного анализа и методы семплирования.  
- Научиться корректно интерпретировать статистические результаты.  
- Подготовиться к дальнейшему изучению оценивания параметров и проверки гипотез.

### 🧭 Ключевые понятия

| Понятие | Смысл |
|----------|--------|
| Генеральная совокупность | Полный набор объектов исследования |
| Выборка | Подмножество объектов для анализа |
| Параметр | Характеристика генеральной совокупности |
| Оценка | Статистическая характеристика выборки |
| Смещение (bias) | Систематическая ошибка оценки |
| Репрезентативность | Способность выборки отражать свойства совокупности |
